Código-fonte / Pré-formatação

Podemos digitar algo em formato de código usando a tag CODE para mudar a font e a tag PRE para formatar.

Exemplo:

        
    fuction(){
        let x;
        if( x % 2 == 0){
            return Par;
        }else{
            return Impar;
        }
    }
    return fuction();
